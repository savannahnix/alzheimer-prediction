# ADNI Multimodal Alzheimer’s Disease Analysis

## Overview
This project integrates genetics, biomarkers, cognition, and MRI data.

## Dataset
- Patients: 3,762
- Observations: 11,631
- Time: 0–36 months

## Key Findings
- APOE + p-Tau217 strongest predictors
- MRI shows structural decline
- MCI is heterogeneous

## Limitations
- MRI coverage limited (~15%)
- Missing biomarker data

## Future Work
- Add imaging models
- Improve imputation
